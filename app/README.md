## 项目目录
## 文件目录
| **目录\文件名称**      |    **说明**   | 
| :-------- | :--------|
| actions  | Redux 公用信息配置 | 
| components | 公共组件及木偶组件 | 
| constants | 定义常量 | 
| containers | 交易页面及智能组件 | 
| fetch |  GET/POST 数据公共定义 | 
| reducers | 定义Redux规则 | 
| router | 路由配置 | 
| static | 项目公用的静态文件 | 
| store | Redux store | 
| util | 通用工具函数 | 
| index.jsx | 默认入口文件 | 
| index.tmpl.html | 入口模板文件 |
